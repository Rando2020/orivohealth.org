# Orivo Health Academy LMS Build Plan

This branch expands the public course catalog into a trackable learning platform.

## Approved defaults

- Public course previews
- Free account required for progress and quizzes
- Email/password and Google authentication
- 80% quiz passing score
- Unlimited retakes with immediate explanations
- Randomized questions from large question banks
- Certificates after final assessment and capstone checklist
- No payment system in the first release
- Vercel hosting with Supabase authentication and Postgres
- ORI-200 APIs, Webhooks, Postman, and Integration Design as the first gold-standard course

## Content standard

Every production course should include:

- Orientation and diagnostic assessment
- Fully written lessons
- Original diagrams and credited visual adaptations
- Guided demonstrations
- Downloadable labs and templates
- Large randomized question banks
- Module quizzes and a final assessment
- Portfolio capstone
- Interview translation

## Added course

ORI-315: PMP, Hybrid Delivery, and Agile Ceremonies covers the July 2026 PMP exam emphasis, predictive, agile, and hybrid delivery, and practical facilitation of delivery ceremonies.
